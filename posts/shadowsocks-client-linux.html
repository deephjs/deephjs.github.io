Linux(Ubuntu)下使用shadowsocks科学上网

记录防踩坑

pip安装shadowsocks（默认已经有python）

#sudo pip install shadowsocks

启动shadowsocks客户端:

新建json文件,名字自定,这里起名为shadowsocks.json，并配置如下:

{
    "server":"{服务端ip}",
    "server_port":{服务端shadowsocks端口号},
    "local_port":{本地shadowsocks客户端端口},
    "password":"{服务端shadowsocks密码}",
    "timeout":{超时(单位:s)},
    "method":"{加密方法}"
}

启动命令：

sslocal -c shadowsocks.json 或
sslocal -c shadowsocks.json #debug模式

安装polipo

本来到这里再把终端代理设置为本地shadowsocks就可以了，但是由于shawdowsocks是走socks 5代理的，需要polipo配合才能翻墙。

polipo安装命令:

#sudo apt-get install polipo

安装好后修改polipo的配置文件/etc/polipo/config：

logSyslog = true
logFile = /var/log/polipo/polipo.log

proxyAddress = "0.0.0.0"

socksParentProxy = "127.0.0.1:1080" #这里端口填本地shadowsocks client的端口
socksProxyType = socks5

chunkHighMark = 50331648
objectHighMark = 16384

serverMaxSlots = 64
serverSlots = 16
serverSlots1 = 32

保存后重启polipo服务：

#sudo /etc/init.d/polipo restart

设置终端代理:

export http_proxy="http://127.0.0.1:8123/"(polipo的默认端口应该是是8123)

测试:

#ping www.google.com

注意事项

  1. shadowsocks集成了服务端和客户端，本教程默认远程（或本地）已开启shadowsocks服务端, 服务端命令是ssserver, 客户端命令是sslocal
  2. 测试系统是Ubuntu14.04,其它发行版linux不保证有效
